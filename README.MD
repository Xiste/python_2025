# 🐍 Introdução à Programação com Python

Repositório com os exercícios e projetos desenvolvidos durante o curso **Introdução à Programação com Python**, ministrado por [Téo Calvo](https://github.com/teocalvo) no Instituto Aaron Swartz.

---

## 📚 Sobre o Curso

Curso voltado para iniciantes em programação, com foco em construir uma base sólida em Python — do conceito de computação até manipulação de dados e desenvolvimento de aplicações simples.

---

## 🧠 Habilidades Desenvolvidas

### 🔹 Fundamentos de Programação
- Conceito de computação: entrada, processamento e saída de dados
- Lógica de programação e pensamento algorítmico
- Configuração do ambiente de desenvolvimento (Python + VS Code)

### 🔹 Tipos de Dados
- **Strings** — manipulação de textos, concatenação e fatiamento
- **Inteiros e Floats** — operações matemáticas e formatação de números
- **Booleanos** — valores `True` e `False` em expressões lógicas
- Python como calculadora: soma, subtração, multiplicação, divisão, potência e módulo

### 🔹 Variáveis e Entrada de Dados
- Declaração e atribuição de variáveis
- Leitura de dados com `input()`
- Conversão de tipos (`int()`, `float()`, `str()`)

### 🔹 Estruturas de Controle
- **Condicionais:** `if`, `elif`, `else`
- **Laços de repetição:** `for`, `while`
- Controle de fluxo com `break` e `continue`
- Tratamento de erros com `try` e `except`

### 🔹 Funções
- Definição de funções com `def`
- Parâmetros, argumentos e retorno de valores
- Boas práticas de modularização de código

### 🔹 Estruturas de Dados
- **Listas:** criação, acesso por índice, modificação, funções `len()`, `append()`, `remove()`, `sort()`
- **Tuplas:** características e diferenças em relação às listas
- **Dicionários:** pares chave/valor, criação, acesso e modificação

### 🔹 Módulos e Bibliotecas
- Importação de módulos com `import`
- Uso de bibliotecas padrão: `math`, `random`, `datetime`

### 🔹 Manipulação de Arquivos
- Leitura e escrita de arquivos em Python
- Trabalho com arquivos CSV

### 🔹 Tópicos Especiais
- Introdução ao **Streamlit** para criação de aplicações web simples
- Consumo básico de **APIs**

---

## 🏋️ Exercícios Praticados

Ao longo do curso foram resolvidos exercícios progressivos, incluindo:

- Programas de saudação e interação com o usuário
- Conversores (segundos → horas/minutos/segundos, raio → área/perímetro)
- Verificador de números pares/ímpares e primos
- Calculadora de fatorial e sequência de Fibonacci
- FizzBuzz
- Detector de palíndromos
- Simulador de sorveteria e loteria
- Tabuada, contador de ocorrências e inversão de palavras
- **Desafio final:** Loteria da Babilônia (com e sem Streamlit)
